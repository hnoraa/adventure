# adventure
An adventure game written in python

## Requirements
* python-3.7.4
(Install Requirements: `pip install -r requirements.txt`)

* pygame==1.9.6
* PyTMX==3.21.7
* six==1.14.0

## Optional Requirements
* Tiled 1.3.2
* paint.net 4.2.9
